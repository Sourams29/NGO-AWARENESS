# NGO Awareness Website

A responsive NGO awareness platform designed to promote social impact initiatives, volunteer engagement, donations, and community outreach activities.

## 🌐 Live Demo

https://inamigos-xi.vercel.app

---

## 📌 Features

- Responsive modern UI
- NGO mission and awareness sections
- Volunteer and community engagement information
- Donation/support sections
- Clean navigation and user-friendly layout
- Mobile-friendly design

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- React
- Vite
- Vercel for deployment

---

## 🚀 Deployment

This project is deployed using Vercel.

Live Website:

https://inamigos-xi.vercel.app

---

## 📂 Project Setup

Clone the repository:

```bash
git clone https://github.com/Abhishiktha2420030792/NGO-Awareness.git
```

Move into the project folder:

```bash
cd NGO-Awareness
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
